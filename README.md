# ZIM4rv
Zero-inflated models to precisely test genetic associations between zero‐inflated count phenotype and rare variants.

## Installation
To install the developer version of ZIM4rv, using the 'devtools' package is easiest.
```
#install.package("devtools")
library(devtools)
install_github("fanx0037/ZIM4rv")
```

## Usage
The following help page will provide quick references for ZIM4rv package and its main functions:
```
library(ZIM4rv)
?zimfrv
```

## References
1. Fan, Q., Sun, S., & Li, Y.‐J. (2021). [Precisely modeling zero‐inflated count phenotype for rare variants](https://doi.org/10.1002/gepi.22438). Genetic Epidemiology, 1–14.
